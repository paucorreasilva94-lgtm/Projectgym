# Project gym

Gym Churn Predictive Analysis
This project aims to predict customer churn in a gym setting using Machine Learning techniques. Through comprehensive data analysis and the training of predictive models, we seek to optimize member retention strategies.

🚀 Project Overview
Customer churn is a critical challenge for any subscription-based business. This project leverages historical data to:

Conduct Exploratory Data Analysis (EDA) to understand member behavioral patterns.

Identify key variables that influence a member's decision to stay or leave.

Train and optimize a classification model to predict churn risk with high precision and recall.

📊 Key Findings
Strongest Predictor: Lifetime is the most powerful indicator of customer loyalty.

Segmentation: Our average member age is 30, making it easier to run targeted campaigns.

Risk: Members with 1-month contracts show the highest churn rate, suggesting a clear opportunity for long-term loyalty plans.

🛠️ Technologies Used
Language: Python

Libraries: pandas, seaborn, matplotlib, scikit-learn

Models Evaluated: Logistic Regression (Selected for its balance and interpretability), Gradient Boosting, Decision Tree, k-NN.

📈 Model Performance
Our Logistic Regression model, optimized via GridSearchCV.

💡 Proposed Retention Strategies
Early-Warning Onboarding: Personalized support during the first 30 days for new members.

Loyalty Conversion: Targeted incentives for short-term contract holders.

Dynamic Re-engagement: Automated interventions when a drop in class frequency is detected.
